# AI Skin Diagnosis Web App

Веб-застосунок для попередньої діагностики захворювань шкіри на основі нейронної мережі.

## Функціонал
- **Завантаження фото:** Користувач завантажує фото ураження шкіри.
- **AI Аналіз:** Використовується модель TFLite (MobilenetV2) для класифікації 7 типів захворювань (Melanoma, Nevus, BCC, etc.).
- **Збереження даних:** Фото та результати зберігаються в базі даних PostgreSQL.
- **Історія:** Перегляд попередніх діагнозів із відображенням збережених зображень.

## Технології
- **Backend:** Python, Flask
- **Database:** PostgreSQL (SQLAlchemy)
- **AI/ML:** TensorFlow Lite
- **Deployment:** Render Cloud
